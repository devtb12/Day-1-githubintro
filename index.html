Commands:
PWD = current path directory
ls = lists files in directory
ls -la = lists file + timestamp
cd = change directories
cd .. = brings you back to the parent level
cd . = lists current path
code . = open up VS Code
cd ~ = goes back to home directory
touch = make file
rm = remove file (note: do not write git rm, just rm <.filename>
mv = creates file

Git Overview:
> git add <.filename> = add file to staging area
> git commit <.filename> = add snapshot of file
> git commit (without -m) = :q to get out of it
> git status = lists file(s) staged or not staged
> git . = adds all files

What the Fork:
1. Fork (adding cloned repo from another source) = menu, top right corner of github repo
2. Click clone (green) button https://github.com/devkel00/learn-git.git and copy URL
3. In the terminal > cd .. (just don't be in an active repo directory)
4. Terminal > git clone https://github.com/devkel00/learn-git.git
Bonus Round:
> git reset = unstage a file
> git rm --cached <./filename> = remove given file from repo
> git remote -v = manage the set of repositories ("remotes") whose branches you track.
> git gitignore = a file that lists file(s) to ignore (don't push API keys)

> git config --global credential.helper 'cache --timeout=28800' // pw timeout is now longer
> git config --global url."https://yourusername@github.com".insteadOf "https://github.com"
